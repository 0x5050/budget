# Build Setup

## install dependencies
Firebase functions
```
$ cd function
$ npm install
```

Nuxt
```
$ cd src
$ npm install
```

## serve
```
$ npm run dev
```

## build nuxt app
```
$ npm run nuxt-build
```

## test nuxt app
```
$ npm run nuxt-test
```

## deploy app
```
$ npm run deploy
```
